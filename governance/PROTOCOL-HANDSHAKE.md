# K-RFS DOIP Handshake Protocol
# K-RFS DOIP 握手协议

## 1. Authentication / 身份验证
**EN:** When a Target aFDO receives an event, it triggers a **Credential Trustworthiness Check**.
**中：** 当目标 aFDO 接收到事件时，它会触发**凭证可信度检查**。

## 2. Logic Protection / 逻辑保护
**EN:** Access to the `Logic Core` is restricted unless the requester's $\Phi$ value meets the sovereignty threshold defined in the AASP.
**中：** 除非请求者的 $\Phi$ 值满足 AASP 定义的主权阈值，否则禁止访问“逻辑核”。

## 3. Apoptosis Trigger / 凋亡触发
**EN:** Unauthorized deep-probing triggers **Digital Apoptosis**, rendering the bit-sequence unreadable.
**中：** 未经授权的深度探测将触发**数字凋亡**，使比特序列变得不可读。
